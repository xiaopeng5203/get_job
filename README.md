# 🍀Get Jobs【Boss直聘自动化投递增强版】

## 项目简介

本项目是一款专为程序员/技术岗位打造的**自动化智能投递工具**，支持 Boss直聘、猎聘、拉勾、智联招聘、51Job 等主流招聘平台。通过 AI 智能打招呼、自动筛选岗位、批量高效投递、企业微信实时推送等功能，极大提升找工作效率和投递命中率。

---

## 主要功能（极致详细拆解版）

### 1. AI 智能打招呼语生成与优化

- **岗位定制化招呼语生成**  
  针对每一个岗位，自动分析职位描述（JD）、岗位职责、任职要求等信息，结合用户的个人简历、项目经验、技能栈等，生成高度个性化、专业化的打招呼语。确保每一次沟通都能精准突出个人与岗位的匹配度，提升HR的关注度和回复率。

- **多轮AI优化与自定义提示词**  
  支持用户自定义AI提示词，灵活调整内容风格。可根据不同岗位需求，突出技术实力、沟通能力、管理经验等。AI会根据用户反馈和历史效果不断优化生成内容，确保每次输出都更贴合实际需求。

- **智能岗位匹配度判断**  
  系统会自动判断岗位与用户经历的匹配度，只有基本符合条件的岗位才会生成招呼语，否则自动跳过，避免无效投递和资源浪费。

- **多平台适配与风格调整**  
  针对Boss直聘、猎聘等不同平台，自动适配招呼语格式和内容风格，确保在各平台均能获得最佳沟通效果。

- **真实案例与模板支持**  
  内置多种真实案例和模板，用户可参考或直接使用，极大降低上手门槛。

---

### 2. 多平台批量自动投递系统

- **一键多平台批量投递**  
  支持Boss直聘、猎聘、拉勾、智联招聘、51Job等主流招聘平台的批量自动化投递。用户只需配置一次，即可实现多平台同步投递，无需手动切换账号或平台。

- **多城市/多岗位关键词循环投递**  
  支持配置多个城市和多个岗位关键词，系统会自动遍历所有组合，循环投递，最大化岗位覆盖率。

- **定时/批量/间隔控制**  
  可自定义每次投递的时间间隔、批量投递数量、定时启动等参数，防止账号被平台风控，保障账号安全。

- **自动切换城市与岗位**  
  系统会根据配置自动切换城市和岗位关键词，智能分批进行投递，提升效率。

- **投递去重与过期控制**  
  内置投递去重机制，支持设置去重的过期天数，防止短期内重复投递同一岗位，提升投递质量。

---

### 3. 智能岗位筛选与黑名单过滤

- **多关键词精准岗位匹配**  
  支持对岗位名称和描述进行多关键词匹配，只要包含任一关键词即可投递，极大提升岗位相关性和命中率。

- **公司/岗位/招聘者黑名单机制**  
  自动维护公司、岗位、招聘者黑名单，支持手动添加和自动合并，避免投递到不合适或有负面记录的企业和岗位。

- **不活跃HR自动过滤**  
  系统会自动检测HR活跃状态，过滤掉长时间未活跃的HR，提升沟通效率和投递成功率。

- **岗位去重与过期控制**  
  支持设置岗位投递去重的过期天数，防止短期内重复投递同一岗位，提升投递效率。

- **灵活配置筛选规则**  
  所有筛选规则均可在配置文件中灵活调整，满足不同用户的个性化需求。

---

### 4. 自动化流程稳定性与异常兜底

- **多重元素定位与重试机制**  
  针对招聘平台频繁的前端变动，系统支持输入框、发送按钮等关键元素的多选择器定位和多次重试，极大提升流程稳定性。

- **智能消息检测与确认**  
  在发送招呼语后，系统会自动检测输入框内容是否清空或聊天区是否出现招呼语，确保消息真正发出后再关闭窗口，避免消息丢失。

- **图片简历上传时序优化**  
  确保图片简历上传和发送逻辑在页面关闭前全部完成，彻底解决因页面提前关闭导致的投递丢失问题。

- **流程异常自动重试与兜底**  
  针对AI生成慢、页面加载慢等异常情况，系统会自动重试，保证整体流程不中断，提升投递成功率。

---

### 5. 图片简历自动发送与格式兼容

- **Boss直聘专属图片简历自动发送**  
  支持Boss直聘平台自动发送图片简历，无需HR主动索要，提升简历曝光率和HR关注度。

- **PDF简历自动转图片**  
  支持将PDF格式简历自动转换为图片（如resume.jpg），兼容主流简历格式，确保简历内容完整展示。

- **多平台图片简历扩展**  
  如有需求，可扩展到其他招聘平台，实现全平台图片简历自动发送。

- **配置简单易用**  
  只需在resources目录下放置图片简历，并在配置文件中开启相关选项即可，无需复杂操作。

---

### 6. 企业微信/Bark 实时推送与多渠道通知

- **企业微信推送**  
  投递结果、异常信息、岗位详情等可实时推送到企业微信群，用户可随时掌握投递进展和异常情况。

- **Bark推送（iOS通知）**  
  支持iOS Bark推送，投递进度、异常等可推送到手机通知栏，第一时间获取重要信息。

- **一键配置多渠道推送**  
  只需填写Webhook或Bark推送地址即可，支持同时开启企业微信和Bark推送，信息同步更及时。

- **推送内容丰富**  
  包括投递成功、失败、异常、岗位详情、AI生成内容等，信息全面，便于复盘和管理。

---

### 7. 高效投递与流程加速

- **等待时间与流程优化**  
  系统会根据实际情况自动优化sleep等待和元素检测时间，整体流程显著提速，提升批量投递效率。

- **自定义参数灵活配置**  
  支持自定义每投递一个岗位的等待时间、元素等待超时等参数，兼顾流程稳定性与效率。

- **批量并发投递支持**  
  支持多线程/批量投递，极大提升大规模投递的效率（如有需求可扩展）。

---

### 8. 易用的集中化配置与详细日志系统

- **集中化配置文件**  
  所有平台参数、筛选规则、AI提示词等均集中在config.yaml文件，结构清晰，易于维护和修改。

- **详细日志输出**  
  系统自动输出详细日志，包括投递流程、异常信息、AI生成内容等，方便用户排查问题和复盘流程。

- **开箱即用体验**  
  用户只需简单配置，即可一键启动，极大降低使用门槛。

- **岗位/公司/HR数据统计与分析**

## 快速开始

1. 克隆仓库  
   ```bash
   git clone https://github.com/xiaopeng5203/get_job.git
   cd get_job
   ```
2. 配置环境（JDK21、Maven、Chrome/ChromeDriver）
3. 修改 `src/main/resources/config.yaml`，填写你的岗位、城市、AI参数等
4. 运行对应平台的主类（如 Boss.java、MobileBoss.java 等）
5. 查看企业微信推送和日志，随时掌握投递进展

详细使用说明请见 [README](./README.md) 及 [doc/](./doc/) 目录。

## 技术亮点

- Playwright 自动化+多重兜底机制，兼容性强，适应招聘平台频繁前端变动

- 配置驱动，支持多平台、多城市、多岗位批量投递，灵活高效
