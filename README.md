# 🍀Get Jobs【Boss直聘自动化投递增强版】

## 项目简介

本项目是一款专为程序员/技术岗位打造的**自动化智能投递工具**，支持 Boss直聘、猎聘、拉勾、智联招聘、51Job 等主流招聘平台。通过 AI 智能打招呼、自动筛选岗位、批量高效投递、企业微信实时推送等功能，极大提升找工作效率和投递命中率。

---

## 近期功能增强与优化（详细版）

### 1. 黑名单功能全面升级

- **支持公司+招聘者组合黑名单**  
  新增"公司+招聘者"黑名单，自动跳过该公司下指定招聘者发布的所有岗位。支持配置有效期（天数）或永久有效，灵活应对不同场景。
- **黑名单配置方式更灵活**  
  所有黑名单（公司、岗位、招聘者、公司+招聘者）均支持两种配置方式：  
  - 字符串方式：如 `"新大陆"`，表示永久拉黑  
  - 对象方式：如 `{name: "公司B", days: 7}`，可设置临时拉黑天数
- **自动拉黑机制**  
  投递岗位后，若收到"不合适"等反馈，系统会自动将该公司+该招聘者组合加入黑名单，并根据配置自动设置拉黑时效，无需手动维护。
- **黑名单持久化与时效管理**  
  黑名单支持持久化存储，自动记录拉黑时间和剩余有效天数，到期后自动恢复投递资格，避免误伤和长期遗漏优质岗位。

### 2. 招聘者信息抓取与格式优化

- **招聘者字段标准化**  
  招聘者信息自动拼接为"真实姓名 | 公司名称 | 岗位"格式，确保数据完整、可追溯。例如：`张三 | 穗彩科技 | HRBP`。若无法获取真实姓名，则用"未知"占位，保证格式统一。
- **岗位职责/任职资格分行输出**  
  抓取到的岗位职责、任职资格等内容，自动按"1、2、3、..."等序号分行输出，每条单独一行，极大提升可读性和后续处理效率。

### 3. 配置文件与反序列化兼容性增强

- **黑名单配置兼容字符串与对象混合**  
  配置文件中的黑名单项可混合使用字符串和对象，系统自动识别并反序列化，无需手动区分。
- **自定义反序列化器**  
  内置自定义反序列化器，确保配置文件的灵活性和兼容性，极大降低配置出错概率。

### 4. 代码健壮性与易用性提升

- **详细注释与示例**  
  配置文件和核心代码均补充了详细注释和示例，帮助用户快速理解和正确配置各项参数。
- **异常处理与日志优化**  
  关键流程增加异常捕获和详细日志输出，便于排查问题和追踪流程。
- **自动导入缺失依赖**  
  自动补全常用 Java 类（如 `File`、`Arrays`）的 import，避免编译报错。

### 5. 其他细节优化

- **岗位详情写入文件格式优化**  
  岗位详情写入 Markdown 文件时，所有字段（岗位名称、公司、地点、薪资、招聘者、职责等）均按标准格式输出，便于后续查阅和分析。
- **兼容多平台黑名单配置**  
  h5boss、job51、lagou、liepin、zhilian 等平台的黑名单配置方式与 boss 完全一致，支持对象和字符串两种方式，配置体验统一。

## 主要功能（极致详细拆解与实现逻辑说明）

### 1. 多平台批量自动投递
- **支持平台**：Boss直聘、猎聘、拉勾、智联招聘、51Job 等主流招聘平台。
- **实现逻辑**：
  - 统一配置入口，所有平台参数集中在 config.yaml 文件，用户只需配置一次。
  - 各平台投递逻辑独立实现，主流程自动遍历所有平台、城市、岗位关键词组合。
  - 每个平台自动登录、自动搜索、自动投递，支持批量、定时、间隔控制，防止账号被风控。
  - 投递去重机制：每次投递前自动检查历史记录，支持设置去重过期天数，避免短期重复投递。
  - 自动切换城市与岗位关键词，循环遍历所有组合，最大化岗位覆盖率。

### 2. AI 智能打招呼与岗位分析
- **岗位定制化招呼语生成**：
  - 自动抓取岗位JD、职责、要求等信息，结合用户简历、项目经验、技能栈。
  - 调用大模型（如GPT-4o）生成高度个性化、专业化的打招呼语。
  - 支持自定义AI提示词，用户可灵活调整内容风格和重点。
  - AI会根据用户反馈和历史效果不断优化生成内容。
- **岗位匹配度判断**：
  - 系统自动分析岗位与用户经历的匹配度。
  - 只有基本符合条件的岗位才会生成招呼语，否则自动跳过，避免无效投递。
- **多平台适配**：
  - 针对不同平台，自动适配招呼语格式和内容风格。
  - 内置多种真实案例和模板，用户可参考或直接使用。

### 3. 智能岗位筛选与黑名单过滤
- **多关键词精准岗位匹配**：
  - 支持对岗位名称和描述进行多关键词匹配，只要包含任一关键词即可投递。
- **黑名单机制**：
  - 支持公司、岗位、招聘者、公司+招聘者组合黑名单。
  - 黑名单支持两种配置方式：字符串（永久拉黑）和对象（可设置有效期days）。
  - 配置示例：
    - 字符串方式："新大陆"（永久拉黑）
    - 对象方式：{name: "公司B", days: 7}（拉黑7天）
  - 自动拉黑机制：投递岗位后，若收到"不合适"等反馈，系统自动将该公司+该招聘者组合加入黑名单，并根据配置自动设置拉黑时效。
  - 黑名单持久化存储，自动记录拉黑时间和剩余有效天数，到期后自动恢复投递资格。
  - 配置文件中的黑名单项可混合使用字符串和对象，系统自动识别并反序列化。
- **不活跃HR自动过滤**：
  - 系统自动检测HR活跃状态，过滤掉长时间未活跃的HR。
- **岗位投递去重与过期控制**：
  - 每次投递前自动检查岗位历史投递记录，支持设置去重过期天数。
- **灵活配置筛选规则**：
  - 所有筛选规则均可在配置文件中灵活调整，满足个性化需求。

### 4. 招聘者信息与岗位详情标准化
- **招聘者信息标准化**：
  - 自动抓取招聘者真实姓名（如 h2.name），公司名和岗位名（如 boss-info-attr）。
  - 拼接为"真实姓名 | 公司名称 | 岗位"格式，若无法获取真实姓名则用"未知"占位。
- **岗位职责/任职资格分行输出**：
  - 抓取到的岗位职责、任职资格等内容，自动按"1、2、3、..."等序号分行输出，每条单独一行。
- **岗位详情写入文件**：
  - 岗位详情写入 Markdown 文件时，所有字段（岗位名称、公司、地点、薪资、招聘者、职责等）均按标准格式输出，便于后续查阅和分析。

### 5. 自动化流程稳定性与异常兜底
- **多重元素定位与重试机制**：
  - 支持输入框、发送按钮等关键元素的多选择器定位和多次重试，极大提升流程稳定性。
- **智能消息检测与确认**：
  - 发送招呼语后，自动检测输入框内容是否清空或聊天区是否出现招呼语，确保消息真正发出后再关闭窗口。
- **图片简历上传时序优化**：
  - 确保图片简历上传和发送逻辑在页面关闭前全部完成。
- **流程异常自动重试与兜底**：
  - 针对AI生成慢、页面加载慢等异常情况，系统会自动重试，保证整体流程不中断。

### 6. 图片简历自动发送与格式兼容
- **Boss直聘专属图片简历自动发送**：
  - 支持Boss直聘平台自动发送图片简历，无需HR主动索要。
- **PDF简历自动转图片**：
  - 支持将PDF格式简历自动转换为图片，兼容主流简历格式。
- **多平台图片简历扩展**：
  - 可扩展到其他招聘平台，实现全平台图片简历自动发送。
- **配置简单易用**：
  - 只需在resources目录下放置图片简历，并在配置文件中开启相关选项即可。

### 7. 企业微信/Bark 实时推送与多渠道通知
- **企业微信推送**：
  - 投递结果、异常信息、岗位详情等可实时推送到企业微信群。
- **Bark推送（iOS通知）**：
  - 支持iOS Bark推送，投递进度、异常等可推送到手机通知栏。
- **一键配置多渠道推送**：
  - 支持同时开启企业微信和Bark推送，信息同步更及时。
- **推送内容丰富**：
  - 包括投递成功、失败、异常、岗位详情、AI生成内容等。

### 8. 高效投递与流程加速
- **等待时间与流程优化**：
  - 系统会根据实际情况自动优化sleep等待和元素检测时间，整体流程显著提速。
- **自定义参数灵活配置**：
  - 支持自定义每投递一个岗位的等待时间、元素等待超时等参数。
- **批量并发投递支持**：
  - 支持多线程/批量投递，极大提升大规模投递的效率（如有需求可扩展）。

### 9. 易用的集中化配置与详细日志系统
- **集中化配置文件**：
  - 所有平台参数、筛选规则、AI提示词等均集中在config.yaml文件，结构清晰，易于维护和修改。
- **详细日志输出**：
  - 系统自动输出详细日志，包括投递流程、异常信息、AI生成内容等，方便用户排查问题和复盘流程。
- **开箱即用体验**：
  - 用户只需简单配置，即可一键启动，极大降低使用门槛。

### 10. 其他辅助功能
- **多平台账号管理**：
  - 支持多平台账号的统一管理和切换，方便用户在不同平台间灵活操作。
- **岗位/公司/HR数据统计与分析**：
  - 自动统计投递岗位、公司、HR的相关数据，帮助用户分析投递效果，优化求职策略。
- **异常自动报警与恢复**：
  - 系统在检测到异常时自动报警，并尝试自恢复，最大限度保障流程不中断。
- **持续更新与社区支持**：
  - 项目持续维护更新，社区活跃，用户可通过QQ群、Issue等方式反馈问题和建议。

## 快速开始

1. 克隆仓库  
   ```bash
   git clone https://github.com/xiaopeng5203/get_job.git
   cd get_job
   ```
2. 配置环境（JDK21、Maven、Chrome/ChromeDriver）
3. 修改 `src/main/resources/config.yaml`，填写你的岗位、城市、AI参数等
4. 运行对应平台的主类（如 Boss.java、MobileBoss.java 等）
5. 查看企业微信推送和日志，随时掌握投递进展

详细使用说明请见 [README](./README.md) 及 [doc/](./doc/) 目录。

## 技术亮点

- Playwright 自动化+多重兜底机制，兼容性强，适应招聘平台频繁前端变动
- AI大模型（支持GPT-4o等）自动生成打招呼语，内容专业、个性化
- 配置驱动，支持多平台、多城市、多岗位批量投递，灵活高效
- 企业微信/Bark机器人推送，信息实时同步，异常及时提醒
- 黑名单自动维护，岗位筛选智能化，避免踩坑和无效投递
- 支持图片简历自动发送，提升简历曝光率和HR关注度
- 详细日志与异常兜底，流程稳定可靠
